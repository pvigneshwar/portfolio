Project Overview

A portable-storage security and monitoring system that combines authentication, Windows USB detection, Android OTG support, real-time access logging, location reporting and VeraCrypt-encrypted storage.

Problem Addressed

Portable USB devices can be lost, stolen or accessed without authorization. Smart USB Guardian creates a shared owner-monitoring workflow across Windows, Android and the web.

My Contribution

Designed the two-client workflow, implemented USB identity and authentication logic, created access monitoring and dashboard features, and integrated encrypted vault access.

PROJECT WORKFLOW

How the system works

1

Register USB

The owner creates an account and registers the pendrive.

2

Generate Identity

A unique usb_guardian.id file is stored on the USB.

3

Create Vault

Private files are placed inside a VeraCrypt encrypted vault.

4

Detect Device

Windows Monitor or Android OTG detects the USB.

5

Activation Event

Device and available location details are reported.

6

Open Login

The USB Guardian authentication screen opens.

7

Validate Access

Granted and denied attempts are recorded.

8

Unlock Vault

The secure vault opens after successful login.

9

Disconnect

Removal updates the dashboard as disconnected.

Key Features

  • Unique USB identity registration
  • Windows USB auto-detection
  • Manual portable launcher
  • Android OTG detection
  • Granted and denied access tracking
  • Activation and security notifications
  • GPS/IP location reporting
  • Connected/disconnected monitoring
  • VeraCrypt-encrypted secure vault
  • Owner dashboard and access-log management
