Project Overview

A secure Java web application that hides and retrieves confidential text using steganography, authentication, role-based access, AES encryption and MySQL database integration.

Problem Addressed

Sensitive information needs protection during storage and exchange. This project combines concealed text, encryption and controlled user access.

My Contribution

Developed the Java web modules, authentication and database integration, and implemented encoding, decoding and encryption workflows.

PROJECT WORKFLOW

How the system works

1

Register User

A user creates an account.

2

Authenticate

The application validates login credentials.

3

Enter Secret

The user supplies confidential text.

4

Encrypt Text

AES encryption protects the message.

5

Encode

Steganography logic conceals the encrypted content.

6

Store Record

The encoded information and metadata are saved.

7

Retrieve

An authorized user opens the decoding workflow.

8

Decode

The concealed content is extracted.

9

Decrypt

The original confidential text is restored.

10

Audit

Admin and report modules support monitoring.
