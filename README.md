# Vigneshwar P — Dark Technical Portfolio

A GitHub Pages-ready personal portfolio with a professional dark theme, lightweight technical graphics and Android-friendly animations.

## Highlights

- Dark navy and cyan technical theme
- One-time opening boot animation
- Animated desktop section guide
- Scroll progress and active navigation
- Lightweight CSS grid/circuit graphics
- Responsive WebP images
- Desktop parallax disabled on mobile
- Reduced-motion accessibility support
- No framework or build step required

## Correct GitHub structure

Upload the files **inside this folder** so `index.html` and `assets/` are at the repository root.

```text
index.html
assets/
alias-x.html
smart-usb-guardian.html
retail-decision-intelligence.html
text-steganography.html
.nojekyll
```

GitHub Pages setting: `main` branch and `/ (root)`.


Latest update: added lighter Android/mobile rendering mode, disabled costly background effects on low-power devices, skipped boot animation on mobile, and changed profile images to object-fit: contain so the top of the head is not cropped.
